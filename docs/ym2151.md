# Yamaha 2151


## Clocks

### Clock A

A 10-bit timer. Can trigger a Key-On in all slots when overflowing (this is "CSM"). From the docs: "At this point it is necessary to stop the interrupt, and there is a control for this as well" üßê

Regsiters CLKA1 and CLKA2

NA = CLKA1 << 2 | CLKA2

Period (ms) = 64 * (1024 - NA) / 3579.545

### Clock B

Period (ms) = 1024 * (256 - CLKB) / 3579.545
